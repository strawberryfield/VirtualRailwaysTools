2.1. How to translate the programs¶

The localization of these programs is made with the GNU GetText Library.

Sample italian locales are distribuited with the source code, in this chapter I will explain how to modify them using Poedit

2.1.1. Required tools¶

The only required tool is Poedit available freely at https://poedit.net/
